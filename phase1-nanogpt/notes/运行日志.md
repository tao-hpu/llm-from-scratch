## 里程碑 3 — `02_attention.py`

A 段（注意力数学本质演示，不训练）：
```
权重矩阵 a（下三角归一化）=
 [[1.0000, 0.0000, 0.0000],
  [0.5000, 0.5000, 0.0000],
  [0.3333, 0.3333, 0.3333]]
输出 c = a @ b =
 [[0., 1.],      位置0 只看自己
  [1., 2.],      位置1 看 0,1 的平均
  [2., 3.]]      位置2 看 0,1,2 的平均
→ 下三角 = 因果掩码；自注意力把"固定平均"换成"由 Q·K 算出来的内容相关权重"
```
